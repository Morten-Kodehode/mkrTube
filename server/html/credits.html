mkrTube

mkrTube is a project created on self initiative. The base idea was initiated by Kjell Harald, and advertised to the members of the Kodehode bootcamp by Jobloop AS.

The purpose of this project is to show that we can take on projects on a whole diffirent level. Communicate with eachother and utilize the tools available to us. The project also display the cooperation of individuals on very diffirent skill levels. How we learn from, and elevate eachother to new heights.
